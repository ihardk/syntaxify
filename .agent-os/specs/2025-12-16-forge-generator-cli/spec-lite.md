# Forge Generator CLI - Lite Summary

Build the `forge` CLI that reads meta component definitions (specs + tokens) and generates production-ready Flutter widgets. Core deliverable: `forge build` command that outputs `app_button.dart` from `button.meta.dart` + `button.tokens.dart`.

## Key Points
- CLI with build/clean/watch commands
- Parse meta definitions from `.meta.dart` files
- Generate Flutter widgets that match manual prototype quality
- File headers with source tracking and checksums
