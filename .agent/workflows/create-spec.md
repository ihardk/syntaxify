# Create Spec Workflow

This workflow uses Agent-OS instructions to create properly formatted specs.

1. Read the Agent-OS spec creation workflow:
```
view .agent-os/instructions/core/create-spec.md
```

2. Follow the Agent-OS process:
   - Step 1: Spec Initiation (from roadmap or user idea)
   - Step 2: Context Gathering (mission-lite, tech-stack)
   - Step 3: Requirements Clarification
   - Step 4: Date Determination
   - Step 5: Spec Folder Creation (YYYY-MM-DD-spec-name)
   - Step 6: Create spec.md
   - Step 7: Create spec-lite.md
   - Step 8: Create technical-spec.md
   - Step 9: Create database-schema.md (if needed)
   - Step 10: Create api-spec.md (if needed)
   - Step 11: User Review

3. Reference product docs:
   - Mission: `.agent-os/product/mission-lite.md`
   - Roadmap: `.agent-os/product/roadmap.md`
   - Tech Stack: `.agent-os/product/tech-stack.md`
