# Create Tasks Workflow

This workflow uses Agent-OS instructions to create task checklists.

1. Read the Agent-OS task creation workflow:
```
view .agent-os/instructions/core/create-tasks.md
```

2. Follow the process to create tasks.md in the spec folder

3. Tasks format:
   - Numbered groups by component
   - Checkboxes for each task
   - Dependencies noted
