# Analyze Product Workflow

This workflow uses Agent-OS to analyze an existing codebase.

1. Read the Agent-OS analyze-product instruction:
```
view .agent-os/instructions/core/analyze-product.md
```

2. Follow the process:
   - Step 1: Analyze existing codebase (structure, tech, progress)
   - Step 2: Gather product context from user
   - Step 3: Execute plan-product with context
   - Step 4: Customize generated documentation
   - Step 5: Final verification and summary

3. Reference meta instructions:
   - Pre-flight: `.agent-os/instructions/meta/pre-flight.md`
   - Post-flight: `.agent-os/instructions/meta/post-flight.md`
