# Execute Task Workflow

This workflow uses Agent-OS instructions to execute spec tasks.

1. Read the Agent-OS task execution workflow:
```
view .agent-os/instructions/core/execute-task.md
```

2. Follow the process:
   - Read spec-lite for context
   - Read tasks.md for task list
   - Execute the next unchecked task
   - Update task checkbox when complete

3. After execution:
```
view .agent-os/instructions/core/post-execution-tasks.md
```
